# 集合



## 有序集合和无序集合

- 有序集合中的元素具有顺序性：基于搜索树实现
- 无序集合中的元素没有顺序性：基于哈希表实现



## 多重集合

存在多重集合，可以重复。但通常提到集合都是指不可重复的集合。



## Set

### Set时间复杂度分析

|                | LinkedListSet | BstSet | BstSet平均 | BstSet最差 |
| -------------- | ------------- | ------ | ---------- | ---------- |
| add（增）      | O(n)          | O(h)   | O(logn)    | O(n)       |
| contains（查） | O(n)          | O(h)   | O(logn)    | O(n)       |
| remove（删）   | O(n)          | O(h)   | O(logn)    | O(n)       |

h 为二分搜索树的高度，n为节点的个数。

二分搜索树可能退化为链表。
