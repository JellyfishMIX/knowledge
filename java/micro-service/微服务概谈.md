## 微服务鉴权

在微服务中，多个服务的无状态化，通常采用两种方案：

- 分布式Session
  - 将用户认证信息存储在共享存储中，通常由用户会话作为key实现简单的hash映射。
  - 当用户访问微服务时，用户数据可从共享存储中获取，用户状态不透明
  - 同时也是一个高可用、易扩展的解决方案。
- OAuth2.0 + Spring Security



## 公共组件

从0开始的项目的多个服务，前期可以把公共组件copy到服务各自代码中，待公共组件稳定后抽离出来，单独做成公共组件服务。
